# The-Blog

## Tech Stack

Nodejs, Express, MongoDB

## Clone the project

```bash
   git clone (https://link)
```

## Create the ".env"

Input the following:

- `access='access-token',`
- `connection="MONGODB_URL",`
- `email=" ",`
- `pass=" "`

## Install dependencies

```bash
   npm install
```

## Start the server

```bash
   npm start
```
